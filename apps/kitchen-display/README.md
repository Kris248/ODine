# ODine Kitchen Display (KDS)

## What this rewrite does

This version turns the KDS into a light, responsive, chef-friendly board that works well on:
- kitchen TV screens
- tablets
- mobile devices

## Main views

- Focus view
- Batch view
- Table view
- Timeline view

## Features

- compact live ticket lane
- one-tap order status changes
- seat-aware order display
- batch grouping by dish + modifiers
- table frequency view
- delayed order highlighting
- responsive layout for TV and mobile

## Development

```bash
cd apps/kitchen-display
npm install
npm run dev
```

## Build

```bash
npm run build
```
